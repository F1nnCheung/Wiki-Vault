# Codex (OpenAI)

Codex 是 OpenAI 推出的云端 AI 编程 Agent，集成于 ChatGPT，能自动从需求分析到部署完成全流程。强调「无人值守」编程——给它一个任务描述，它克隆仓库、写代码、跑测试、提交 PR。

## 核心定位

全流程自动化编程 Agent——最接近「AI 独立完成项目」的工具。

## 核心能力

- 云端独立运行，不占用本地资源
- 从需求描述到部署的全流程自动化
- 克隆 Git 仓库 → 分析代码 → 修改 → 提交 PR
- 支持多文件、跨模块的大规模改动
- GPT 系列模型支持

## 适用场景

- ✅ 自动化脚本和批量处理
- ✅ 代码迁移和重构
- ✅ 从零搭建新项目
- ❌ 需要精细控制的复杂业务逻辑
- ❌ 与现有项目深度集成的增量开发

## 定价

- Pro：$20/月
- Plus：$200/月（更多 Token）

## 与 Claude Code 对比

| 维度 | Codex | Claude Code |
|---|---|---|
| 运行方式 | 云端 | 本地终端 |
