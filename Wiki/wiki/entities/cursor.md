# Cursor

Cursor 是 Anysphere 开发的 AI 原生 IDE（基于 VS Code），2026 年估值 290 亿美元，是 AI Coding 领域最大的创业独角兽。

## 核心定位

全能均衡型 AI IDE——既有传统 IDE 的完整功能，又有 AI 深度整合。

## 核心能力

| 能力 | 说明 |
|---|---|
| **Tab 补全** | <100ms 延迟，多行代码预测 |
| **Cmd+K 内联编辑** | 选中代码后自然语言修改 |
| **Composer** | 跨文件编辑，同时修改多个文件 |
| **Background Agents** | AI Agent 在云端后台运行，不阻塞 IDE |
| **多模型切换** | 支持 Claude、GPT、Gemini 等 |

## 版本演进

- **Cursor 1**：VS Code fork + AI Chat
- **Cursor 2**：Composer + Tab 补全 + Agent
- **Cursor 3**（2026.04）：多智能体协同工作区

## 定价

- Free 版：基础功能
- Pro：$20/月
- Business：$40/用户/月
- Enterprise：定制
