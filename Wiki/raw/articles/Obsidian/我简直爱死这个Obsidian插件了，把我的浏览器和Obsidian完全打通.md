蒋先森 *2026年3月29日 19:16*

用 Obsidian 的人都知道它有多强。

本地存储、双向链接、Markdown 格式、配上 AI 之后更是直接起飞——我所有的笔记、复盘、灵感碎片全都在里面，Claude Code 可以直接读取我的知识库，帮我理解、总结、生发新想法。这套组合我用了很久，顺得不行。

但它有一个痛点，用过的人都懂：

**外面的信息，不好进来。**

Obsidian 是本地知识库，只认 Markdown 格式。你在网上刷到一篇好文章，想存进去，怎么办？

复制粘贴？格式全乱，图片丢失，链接断掉，粘进去的东西根本没法看。

就算你有方法可以解决掉，但是就需要手动操作这个事，我就有点接受不了。

现在都啥时代了，我早就已经习惯指挥AI干活，我在一旁当监管了。

所以保存外部信息这个事情，看起来是件小事，但是因为这个节点没跑通，导致我的整个工作流无法闭环。

这个链路，我卡了很久。直到遇到了一个叫 **Obsidian Web Clipper** 的浏览器插件。

**这个插件就做一件事，但做到了极致**

它是谷歌浏览器的一个插件，安装只需要去 Chrome 应用商店搜索 "Obsidian Web Clipper"，点安装，完成。

它只有一个功能： **把你当前网页的内容，一键保存为 Obsidian 笔记。**

就这一个功能，彻底改变了我的信息收集方式。

安装好之后，操作流程是这样的：

**第一步：安装插件**  
打开 Chrome 网上应用店，搜索"Obsidian Web Clipper"，点击"添加到 Chrome"。

**第二步：选择保存内容**  
在任何你想保存的网页中，点开插件图标，一键保存。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

**第三步：本地笔记**  
然后会授权让你打开Obsidian，点确认。这篇文章就变成了你 Obsidian 里的一条笔记。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

全程不超过 3 秒。

**有了这个插件之后，我的浏览器直接和我的Obsidian打通了！**

我刷到一篇写得很好的公众号，用微信里的"在浏览器中打开"打开这篇公众号，点一下 Web Clipper——整篇文章，标题、正文、图片，全部完整保存进 Obsidian，格式干净，像一篇正经笔记。

然后我可以直接让 Claude Code 帮我阅读这篇文章，提炼观点，生成我自己的笔记。 **从"看到"到"变成自己的知识"，全链路打通了。**

**场景二：YouTube 视频**

这个才是真正炸裂的功能。

打开一个 YouTube 视频，点 Web Clipper，保存。

你以为它只保存了一个链接？不—— **它把这个视频直接嵌入到 Obsidian 笔记里，你可以在 Obsidian 内部直接播放这条视频。** 同时，笔记里还自动带上了视频的标签、摘要信息。

然后你可以打开 Claude Code，说"帮我总结这个视频"，AI 直接开干。

以前我在 YouTube 上收藏了几十个教程视频，从没有认真看完过一个。现在，它们全部进了我的知识库，变成了可以被 AI 处理的素材。

**信息收集的链路，终于闭环了**

装上这个插件之后，我的信息流是这样的：

看到好内容 → 一键 Clip 进 Obsidian → Claude Code 协助消化 → 生成自己的笔记 → 沉淀进知识库

以前收藏是终点，现在收藏是起点。

我估摸着，这一个插件，让我的信息利用率提升了不止十倍。不是夸张，是那种"以前的收藏等于白收藏"的真实反差。

**工具本身不值钱，改变信息和你的关系，才值钱。**

如果你也在用 Obsidian，这个插件是我目前推荐的第一个必装插件。免费，极简，就一个功能，

但那一个功能，补上了我知识体系里最大的漏洞。
