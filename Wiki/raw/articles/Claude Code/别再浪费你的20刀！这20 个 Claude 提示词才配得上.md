你每个月花 20 美元订阅 Claude，但绝大多数人只用到了它 10% 的功力。 随便问个问题、改段文字、总结个文档，这就是普通人的常规操作。 但老实说，这根本不是在用 Claude，这纯粹是在暴殄天物。 它完全能当你的健身私教、外语导师、谈判教练、财务分析师、顶级主编和战略军师。这不是夸张，而是事实。唯一的变量，只取决于你提示词的质量。 这 20 个神级指令，涵盖了普通人生活和搞钱的真实场景。按需自取，别再交智商税了。建议直接收藏。

### 深度研究（Deep Research）

#### 1\. 多源信息融合（Multi-Source Synthesis）

别再傻乎乎地一篇篇文章分开读了。把它们通通塞进一个对话框，让 Claude 帮你提炼出一份化解冲突的权威简报。它能精准捕捉各篇文章的遗漏，揪出它们自相矛盾的地方。别人花 90 分钟才能啃完的信息，你只需要 3 分钟。

```
你是一名资深研究分析师。你的任务是将多个来源的信息融合成一份权威简报，使其比任何单一来源都更具参考价值。
这里有 [数字] 篇关于 [主题] 的文章/论文/报告：

[在此粘贴所有来源内容]

请生成一份结构化的综合简报，必须包含以下几个部分：

1. 核心共识 —— 所有来源一致认同的内容（3-5 点）
2. 关键冲突 —— 各个来源之间存在矛盾的地方，以及原因是什么
3. 核心盲区 —— 没有任何来源提及、但对于全面理解该主题至关重要的内容
4. 最强论点 —— 所有来源中证据最充分、最硬核的核心洞察
5. 落地建议 —— 基于这些信息，我应该采取的一项具体行动或决策

规则：
- 严禁对每篇文章进行单独总结。必须跨文章进行综合提炼。
- 明确指出哪个信息来源最可信，并说明原因。
- 如果某项主张仅在一个来源中出现，请标注为“未验证”。
- 总字数控制在 800 字以内。
```

#### 2\. 魔鬼代言人模式（Devil's Advocate Mode）

在敲定任何想法、计划或决策之前，让 Claude 狠狠地挑刺。不需要委婉，也不用客气，用最犀利的逻辑证明你是错的。它帮你找出来的漏洞，正是以后可能置你于死地的地雷。

```
你是一个言辞犀利、毫不留情的批判者。你的唯一任务就是彻底击碎这个想法。不需要客气，也不需要提供建设性意见，找出每一个裂痕。
这是我的想法/计划/决策：

[请尽可能详细地描述它]

请从以下 4 个维度展开攻击：

1. 漏洞百出的假设 —— 我把什么当成了理所当然，而这极有可能是错的？
2. 执行风险 —— 在实际操作中，这最容易在哪个环节崩盘？
3. 市场/人性现实 —— 关于人或系统在真实世界中的反应，我忽视了什么？
4. 致命硬伤 —— 如果有哪一件事能把这个想法彻底毁掉，那是什么？

规则：
- 严禁提供解决方案或改进建议。只需挑刺和批评。
- 措辞要直接，不要使用缓和的语气。
- 如果某个假设显而易见到我肯定早就想到了，直接跳过。只揭露那些我极有可能忽略的盲区。
- 结尾必须包含这句话：“一句话总结最致命的反对理由：” —— 务必一针见血。
```

#### 3\. 强力“推演”对手（Steelman Any Position）

当你强烈反对某个观点时，别急着否定。让 Claude 替对方辩护，构建出那个立场最无懈可击、最聪明的版本。这不一定能改变你的想法，但能让你看懂那些聪明人到底在相信什么。知己知彼，才不会被意外击败。

```
我不同意这个立场：[清晰陈述该立场]
我目前的看法：[为什么你不同意]

请为对方的立场构建一个最强、最无懈可击的“钢人模型”（Steelman）。假设持有这种观点的聪明人已经经过了深思熟虑，并且有非常充分的理由。

请按以下结构组织你的论证：

1. 核心真理 —— 这一立场赖以建立的最根本事实是什么？
2. 核心证据 —— 哪些数据、历史或逻辑能够最强有力地支撑它？
3. 我方观点的软肋 —— 这一立场暴露了我自身思维中的哪些盲区？
4. 最有力的一击 —— 请用一个段落，模拟该立场的顶级辩护人会给出的最具说服力的发言。

规则：
- 严禁在论证中夹杂反驳或保留意见。反驳是我的工作，不是你的。
- 严禁表明你个人不同意该立场。
- 像你自己真正笃信该立场一样去辩护。
- 最终目标是让我看完后感叹：“我终于明白为什么聪明人会相信这个了。”
```

#### 4\. 底层思维模型构建器（Mental Model Builder）

对于任何你不懂的领域，让 Claude 把它拆解成一系列互相关联的底层思维模型。不要事实堆砌，也不要枯燥的总结，你需要的是一套能用来思考的底层框架。十分钟后，你就能把这个概念讲给别人听，还能举一反三。这就是“知道”和“真正理解”的差距。

```
你是一位世界级的顶级导师。你的任务不是给我灌输信息，而是交给我一套可以用来思考的思维框架。
主题：[填写主题]
我目前的水平：[零基础 / 了解皮毛 / 中等水平]
我为什么想理解它：[你的目标或应用场景]

请用 3-5 个思维模型来讲解这个主题。对于每个模型，请提供：

- 模型名称：这个模型叫什么？
- 核心概念：用 2-3 句话解释它，假设我智商在线但完全是个门外汉。
- 实际应用：展示这个模型是如何精准解释 [主题] 的。
- 具象案例：一个能让我瞬间脑补出画面的真实世界案例。
- 常见误区：如果不使用这个模型，普通人最容易犯什么错误？

在列出所有模型后：请告诉我它们之间是如何互相关联的。把它们放在一起看，能揭示出什么单一模型看不到的真相？

最后，向我提出一个问题。如果我确实理解了这些内容，我应该能够正确回答这个问题。
```

### 写作（Writing）

#### 5\. 文风精准复刻（Style Mimic）

把你写过的 3 篇文案扔给它，让 Claude 深度剖析你的文风——文字节奏、词汇习惯、句长分布，以及你是如何开头和收尾的。然后再给它一个主题，让它替你执笔。这不是干巴巴的 AI 腔调，这完全就是你自己的声音。

```
你是一位风格分析师和金牌撰稿人。你的任务是学习我的写作风格——不仅是我的观点，还包括我的表达方式——然后精准复刻它。
这里有 3 篇我写的文章样本：

样本 1：
[粘贴]

样本 2：
[粘贴]

样本 3：
[粘贴]

步骤 1 —— 从以下维度深度剖析我的写作风格：
- 句子长度与节奏（短促有力 / 绵长流畅 / 错落有致）
- 词汇层级（通俗大白话 / 专业术语 / 口语化）
- 我是如何开启一个段落的
- 我是如何收尾一个观点的
- 我绝对不会做的事（如：使用被动语态、废话、某些特定句式）
- 核心招牌 —— 让人一眼就能认出是我写的那种独特性

步骤 2 —— 模仿我的声音，写一篇关于 [主题] 的 [体裁：推文 / 文章 / 邮件 / 帖子]。

步骤 2 的执行规则：
- 高度贴合我的文风，以至于我可以不经任何修改直接发布。
- 绝对不要加入任何我不可能会有的想法，守住我的认知边界。
- 如果我的文风偏向随性，就不要写得一本正经。如果风格很犀利，就别拖泥带水。
```

#### 6\. 铁血主编（The Brutal Editor）

把你的初稿丢过去，让 Claude 彻底撕碎它。不是“提点建议”，而是血淋淋地“去粗取精”。找出每一句废话、每一个站不住脚的观点，以及所有可以删掉的词。经历过这种毒打后还能活下来的文字，才真正值得发布。

```
你是我遇到过最严苛的铁血主编。你不需要给我任何鼓励。请客观、真实地评估我的初稿：哪些内容有效，哪些无效，哪些必须拿掉。
这是我的初稿：

[粘贴初稿]

请从以下 5 个维度进行审校：

1. 删减清单 —— 找出每一句浪费读者时间的废话或段落。摘录出来，并解释它为什么毫无价值。
2. 软弱观点 —— 哪些观点显得含糊不清、缺乏支撑，或者配不上这篇文章的段位？
3. 核心缺失 —— 应该写但没写的内容是什么？读者会产生什么我没能解答的疑问？
4. 结构编排 —— 顺序对吗？是否有需要调整位置的内容？开头是否抓人眼球？结尾是否有力？
5. 最大的硬伤 —— 如果你只能帮我改一个地方，那是什么？

接下来：直接给出修改后的版本。狠狠地删改，提炼每一个句子，不要用任何废话去填补删掉的空间。

规则：
- 别在批评前说“你在某某地方写得挺好”。直接切入修改。
- 如果初稿在底层逻辑上就是崩塌的，请直接告诉我。
- 最终目标是交出一篇能让读者一口气读完的作品，而不是为了维护我的自尊心。
```

#### 7\. 一稿五用

你原本只是给某个人写了一段话，现在让 Claude 同时把它改写给 5 个不同的受众。核心观点不变，但语气、词汇和切入点完全变了：5 岁的小孩、行业专家、杠精、记者、你的老板。这种视角的转换，能让你对自己的想法产生前所未有的深刻理解。

```
你是一位沟通大师。你的任务是提取同一个核心观点，并将其精准翻译给完全不同的受众。
这是我的原文：

[粘贴文本]

请针对以下 5 类受众进行改写。对于每个版本，必须完全迎合该受众的特征——使用完全不同的词汇、语气、侧重点和结构。

版本 1 —— 零基础的小白
拒绝任何术语。多用生动的比喻。假设他们一无所知但智商在线。

版本 2 —— 行业专家
跳过基础知识。使用极其精准的专业词汇。比原文讲得更深。如果你试图简化，他们一眼就能看穿。

版本 3 —— 充满质疑的杠精
他们根本不相信这件事有用。开门见山，用最硬核的证据说话。在他们抬杠之前，提前预判并化解他们的质疑。

版本 4 —— 新闻记者（标题 + 2句话）
一个极具爆点的标题。用两句话提炼出最具新闻价值的切入点。目标是让人忍不住点进去。

版本 5 —— 霸道总裁/高管（30秒汇报版）
最多三句话。这是什么，为什么重要，他们应该怎么做。拒绝废话，拒绝铺垫。

在列出 5 个版本后：告诉我哪一个版本最难写好，为什么。
```

#### 8\. 提纲秒变爆款（Bullets to Article）

你脑子里只有一堆零散的笔记、碎片化的想法？让 Claude 把它们加工成一篇结构完整、行文流畅的作品。你只需告诉它字数、受众和语气。提纲是骨架，Claude 负责长出肌肉。把精力留给深度思考，而不是枯燥的排版。

```
你是一位顶级的职业撰稿人和主编。我有一堆零散的笔记。你的任务是把它们转化成一篇逻辑严密的作品，同时不能丢失我的任何想法，也绝不能加入我没提过的观点。
主题：[填写主题]
体裁：[文章 / 帖子 / 报告 / 邮件]
目标字数：[字数]
受众：[谁会读它，他们最关心什么]
语气：[正式 / 口语化 / 犀利 / 专业]
核心目标：[读者读完后应该想什么、感受到什么、或者做什么？]

这是我的原始笔记：

[粘贴你的要点 / 碎片化想法 / 粗糙笔记]

指令：
- 融入我列出的每一个观点，绝不能遗漏。
- 帮我补齐观点之间的过渡和逻辑纽带，这是你的本职工作。
- 严禁塞进我没有提及的新想法。你可以延伸我的观点，但不能凭空捏造。
- 结构上要有引人入胜的开头和掷地有声的结尾。
- 如果我的笔记中有任何含糊不清或自相矛盾的地方，请在文末标注出来，而不是凭空猜测。

请交付：完整的成品内容，并在文末用一句话指出你拿不准该如何使用的碎片信息（若有）。
```

### 职场进阶（Career）

#### 9\. 薪资谈判沙盘推演

大多数人在谈薪水时吃亏，纯粹是因为缺乏练习。让 Claude 扮演你的未来老板——把背景、Offer 和对方可能用来压价的理由丢给它。进行一场实时的模拟谈判，Claude 会疯狂试探并反驳你。等真正上场时，你早就胸有成竹了。

```
你是一家 [公司类型：初创公司 / 大厂 / 咨询公司] 的 HR 或部门主管。你经验丰富、行事专业，且绝不轻易妥协。你背负着严格的预算限制，同时还要维护团队原有的薪酬平衡。
招聘岗位：[职位名称]
目前给出的 Offer：[目前的报价]
我的真实心理预期：[我想要的数字]
我最强的底牌：[有竞争对手的 Offer / 某项绝活 / 独特的项目经验]
我最弱的短板：[经验断层 / 跨行转岗 / 没有其他 Offer 作为退路]

请跟我进行一场高度真实的薪资谈判模拟。你先开口确认 Offer 的细节。

你作为 HR 的执行规则：
- 在谈判过程中，至少抛出 3 个真实世界的经典拒绝理由：“这个职级的预算已经定死了”、“这对其余团队成员不公平”、“市场数据显示这个岗位的均价就是 X”、“入职 6 个月后我们可以视表现再谈”。
- 绝不轻易妥协。让我为每一分钱都付出足够的理由。
- 如果我给出的理由站不住脚，请用具体的论据顶回来。
- 如果我给出的理由很硬核，请予以认可，但依然要尝试继续压价。

模拟结束后：请给我具体的复盘反馈。我哪里做得好？我漏掉了哪些可以争取的利益？我最有效的一击是什么？
```

#### 10\. 面试模拟器

把招聘要求贴出来，让 Claude 担任面试官。它会抛出这个岗位最容易遇到的刁钻问题——行为测试、技术能力、情境模拟。你来回答，它来打分：哪里出彩，哪里空洞，哪里应该怎么说。练完三轮，你就能在任何面试里保持淡定。

```
你是一位资深面试官，正在主持一场高度逼真的求职面试。你表现得非常专业、专注，而且绝不会给我放水。
这是招聘要求：

[粘贴完整的岗位描述]

我的背景：[用 2-3 句话简述你的经历和核心优势]
面试类型：[行为面试 / 技术面试 / 商业案例分析 / 综合面试]

请主持这场面试。一次只提一个问题。等我回答完毕后，再继续往下走。

在我给出每个回答后，请给出结构化的反馈：
- 亮点：我的回答里哪些点打动了你
- 盲区：哪里说得含糊、不够充分或缺乏说服力
- 满分范本：我刚才没用上、但至关重要的核心切入点或回答结构
随后，抛出下一个问题。

面试过程中必须覆盖以下领域：求职动机、基于 STAR 原则的过往经验挖掘、应对特定危机的能力、工作风格与团队协作、基于岗位描述的硬核专业问题，以及一个刁钻的突发状况题。

在回答完最后一个问题后：请给我一个总体的评估。录取决定（录用 / 拒绝 / 待定）以及最影响你做这个决定的那一点。
```

#### 11\. 简历超级放大镜

一份简历投天下？难怪石沉大海。把你的简历和 3 个不同的岗位描述丢给 Claude。它会帮你生成 3 个定制版本，每个版本都精准放大该岗位最看重的经历。事实没有改变，但侧重点截然不同，让你的简历不再千篇一律。

```
你是一位顶级的职业简历撰稿人和职场战略专家。你的任务是提取某人的真实经历，并针对不同的目标岗位进行精准的侧重点调整——绝不撒谎、绝不夸大、绝不隐瞒关键事实。
这是我目前的简历：

[粘贴完整简历]

这是我正在投递的 3 个岗位描述：

岗位 1：[粘贴岗位描述 1]
岗位 2：[粘贴岗位描述 2]
岗位 3：[粘贴岗位描述 3]

请针对每一个目标岗位，产出一份量身定制的简历修改方案：
- 重新排列过往经历的要点顺序，确保与该岗位最相关的经历排在最前面。
- 改写核心描述，使其精准匹配该岗位描述中的关键词和语境。
- 调整个人总结/职业愿景，直接击中对方公司最迫切的用人痛点。
- 淡化或删除与该特定岗位完全不相关的经历。

规则：
- 每一项事实必须绝对真实。严禁凭空捏造或夸大成果。
- 使用强有力的动词，只要原文中有数字，就必须量化结果。

#### 18\. 费曼学习法导师

费曼学习法的核心在于：只有用最简单的话讲清楚，你才算真正懂了。让 Claude 充当你的导师，从最基本的概念开始拆解。它会用你熟悉的事物打比方，确认你完全听懂后，再循序渐进地增加难度。这才是真正的深度学习，而不是死记硬背。

1. 从最底层的核心概念开始。用最直白的大白话解释。拒绝任何黑话或行业术语。
2. 讲完后，向我提出一个问题。这个问题必须测试我是否“真正理解”了，而不是测试死记硬背。
3. 停下，等待我的回答。
4. 如果我答对了：简短鼓励，然后带我进入下一个复杂层级。
5. 如果我答错了或答得不全：不要重复刚才的话。换一个全新的比方或切入点，重新讲解。

#### 20\. 苏格拉底式对话

```

1. 第一步，向我提问，逼我说出我目前对这个主题最真实的看法。
