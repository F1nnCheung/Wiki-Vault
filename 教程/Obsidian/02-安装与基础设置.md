# Obsidian 02 — 安装与基础设置

> 从下载到第一个 Vault，从基础设置到主题美化。本章帮你完成 Obsidian 的「地基」搭建。

---

## 安装

### 下载

访问 [obsidian.md](https://obsidian.md) 下载对应平台版本：

| 平台 | 方式 |
|---|---|
| **Windows** | 官网下载 `.exe` 安装包 |
| **macOS** | 官网下载 `.dmg` 或 App Store |
| **Linux** | AppImage / Snap / Flatpak |
| **iOS** | App Store |
| **Android** | Google Play 或官网 APK |

### 创建第一个 Vault

Vault 就是你的知识库——本质上是电脑上的一个文件夹。

```
第一步：打开 Obsidian → "Create new vault"
第二步：输入 Vault 名称（如 "MyBrain"）
第三步：选择存储位置（建议 ~/Documents/MyBrain）
第四步：点击 Create
```

> 💡 **Vault 位置建议**：选一个你能完全控制的本地文件夹。不要放在云盘同步目录（iCloud/OneDrive 可能产生冲突）。后续用 Git 做同步，见 [[07-Git同步与备份]]。

## 基础设置

安装后建议立即调整以下设置（Settings → 各选项卡）：

### 编辑器

| 设置项 | 推荐值 | 原因 |
|---|---|---|
| 显示行号 | ✅ 开启 | 方便引用和定位 |
| 严格换行 | ✅ 开启 | Markdown 标准行为 |
| 默认编辑模式 | 实时预览 | 所见即所得 |
| 拼写检查 | ✅ 开启 | 减少笔误 |

### 文件与链接

| 设置项 | 推荐值 | 原因 |
|---|---|---|
| 自动更新内部链接 | ✅ 开启 | 重命名笔记时自动更新所有引用 |
| 新建笔记的存放位置 | Inbox（见下文） | 所有新笔记先进收件箱 |
| 附件存放位置 | `assets/` | 图片和附件统一管理 |

> ⚠️ **Wiki 链接 vs Markdown 链接**：初学者保持默认的 `[[Wiki链接]]` 即可。进阶用户如需跨工具兼容，可将「使用 Wiki 链接」关闭，改用标准 `[文本](路径.md)` 格式。

### 外观

| 设置项 | 推荐 |
|---|---|
| 基础颜色 | 暗色（护眼）/ 亮色（白天） |
| 字体 | 默认即可，或安装 JetBrains Mono |

## 核心插件（免费内置）

Obsidian 内置了大量核心插件，建议**立即开启**以下 4 个：

| 插件 | 功能 | 为什么重要 |
|---|---|---|
| **图谱视图** | 可视化笔记关联 | 发现知识间的隐藏联系 |
| **反向链接** | 查看哪些笔记引用了当前笔记 | 不被链接的单向性蒙蔽 |
| **标签面板** | 按标签浏览笔记 | 多维度导航 |
| **大纲** | 显示当前笔记的标题结构 | 快速跳转 |

开启路径：Settings → Core plugins → 切换开关。

## 主题推荐

Obsidian 的社区主题市场有数百种选择。推荐两个：

### Minimal（最受欢迎）

```
Settings → Appearance → Themes → Browse
搜索 "Minimal" → Install and use
```

- 极简设计，不分散注意力
- 高度可定制（配合 Style Settings 插件）
- 手机端表现优秀

### Baseline（2026 年新星）

2024 年官方年度最佳主题 Cupertino 的作者打造。特点：
- 默认状态就好用，不需花时间调参数
- 兼容 Minimal 绝大多数 CSS 功能类
- 在一个主题内集合了多种配色和工作区布局
- 手机端单独调优，非桌面缩小版

## 快捷键速查（macOS / Windows）

| 操作 | macOS | Windows/Linux |
|---|---|---|
| 快速切换笔记 | `Cmd + O` | `Ctrl + O` |
| 命令面板 | `Cmd + P` | `Ctrl + P` |
| 打开图谱视图 | `Cmd + G` | `Ctrl + G` |
| 切换编辑/预览 | `Cmd + E` | `Ctrl + E` |
| 插入链接 | `Cmd + K` | `Ctrl + K` |
| 搜索当前文件 | `Cmd + F` | `Ctrl + F` |
| 全局搜索 | `Cmd + Shift + F` | `Ctrl + Shift + F` |

## 验证清单

安装配置完成后，做以下验证：

```
□ 创建了一个 Vault，放在本地文件夹
□ 开启行号显示和严格换行
□ 开启了图谱视图、反向链接、标签面板、大纲
□ 安装了一个主题
□ 创建了第一篇测试笔记
□ 在测试笔记中输入 [[ 测试了链接自动补全
□ 打开图谱视图，看到了你的第一个节点
```

## 本章小结

1. 从 obsidian.md 下载，创建本地 Vault
2. 立即开启 4 个核心插件：图谱、反向链接、标签、大纲
3. 基础设置：行号 + 严格换行 + 附件统一放 assets/
4. 主题推荐 Minimal（最稳）或 Baseline（最美）
5. 用快捷键 `Cmd+O` 和 `Cmd+P` 提高效率
