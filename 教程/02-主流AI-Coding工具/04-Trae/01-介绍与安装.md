# 2.4.1 Trae 介绍与安装

> Trae 是国内开发者首选：永久免费、中文理解 98%、国内直连极速。如果你是国内开发者且预算为零，Trae 是最佳起点。

---

## 概述

Trae 是字节跳动推出的 AI 原生 IDE，2026 年中国市场份额 41.2%，注册用户突破 600 万。内置豆包、DeepSeek、GLM、Qwen 等国产顶级模型。

**官网**：trae.cn（国内）/ trae.ai（国际）

> ⚠️ 重要：认准 `.cn` 域名——国内版和国际版是不同的产品，国内用户用 trae.cn。

---

| 特点 | 说明 |

## 安装

### 系统要求

| 系统 | 要求 |
|---|---|
| Windows | Windows 10/11，64 位 |
| macOS | 12.0+（Intel / Apple Silicon） |
| Linux | Ubuntu 20.04+ / Debian 11（.deb） |

### 安装步骤

1. 访问 **https://www.trae.cn**（⚠️ 认准 `.cn` 域名）
2. 点击「下载 IDE」，安装包约 100MB
3. 安装后首次启动
4. **用手机号**或**稀土掘金账号**登录

### 从其他 IDE 迁移

Trae 支持一键导入配置：

```
设置 → 从其他编辑器导入
  ├─ VS Code
  ├─ Cursor
  └─ 手动导入配置文件
```

导入内容包括：设置、快捷键、已安装插件、主题。

## 初次配置

安装完成后，建议完成以下初始化：

### 1. 选择默认模型

```
设置 → 模型配置
  ├─ 豆包（默认，推荐日常使用）
  ├─ DeepSeek（代码推理能力强）
  ├─ GLM（综合能力强）
  └─ Qwen（阿里生态）
```

### 2. 创建项目规则

在项目根目录创建 `.trae/rules.md`：

```markdown
# 项目规则

- 使用 TypeScript 严格模式
- 组件使用 React 函数组件 + Hooks
- 样式使用 Tailwind CSS
- 测试使用 Vitest
```

### 3. 导入 VS Code 配置（可选）

如果有 VS Code 配置，一键导入，无需重新设置。
