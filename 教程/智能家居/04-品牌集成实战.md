# 智能家居 04 — 品牌集成实战

> 这一章是一线实战——逐一攻破米家、海尔、美的、海信、Apple HomeKit 五大品牌的接入。建议从简单的开始练手，最后攻克最棘手的海尔。

---

## 总览：五大品牌接入对比

| 品牌 | 集成名称 | 获取方式 | 认证方式 | 难度 | 覆盖设备 |
|------|----------|----------|----------|------|----------|
| 海信 | Hisense | 离线安装包 | 账号 + 密码 | ⭐ | 电视、空调（含科龙） |
| 美的 | Midea AC LAN | HACS 直接搜 | 账号 + 密码 | ⭐ | 空调、洗衣机、热水器（含 colmo） |
| 米家 | Xiaomi Miot | HACS 搜索下载 | 账号 + 密码 + 验证码 | ⭐⭐ | 上百种小家电 |
| HomeKit | HomeKit Bridge | HA 内置 | 扫码配对 | ⭐⭐ | 所有已接入 HA 的设备 |
| 海尔 | Haier | 离线安装包 | **Token 抓包** | ⭐⭐⭐ | 冰箱、洗衣机、热水器（含卡萨帝） |

> ⚠️ **重要：品牌兼容链规则**
>
> 高端子品牌需先导入母品牌 APP：
> - 卡萨帝（海尔高端）→ 先导入「海尔智家」APP
> - colmo（美的高端）→ 先导入「美的美居」APP
>
> 海尔兼容卡萨帝，但卡萨帝不兼容海尔；美的兼容 colmo，但 colmo 不兼容美的。

---

## 一、米家（Xiaomi Miot）⭐⭐

米家绝对是智能家居设备的主力。虽然大家电不多，但小家电上百种控制功能——灯、插座、传感器、扫地机器人、空气净化器等。

### 安装步骤

1. **HACS 搜索**：侧边栏 HACS → 搜索 `xiaomi` → 找到 Xiaomi Miot
2. **下载**：点击进入 → 右下角 Download → 等待下载完成
3. **重启**：系统提示重启 → 在 Container Manager 中重启 HA 容器
4. **添加集成**：设置 → 设备与服务 → 添加集成 → 搜索 `Xiaomi Miot`
5. **选择集成方式**：推荐「自动发现」
6. **登录**：输入米家账号 + 密码
7. **验证码**：点击蓝色链接获取验证码 → **回填到本页面**（不是链接打开的页面）
8. **选择设备**：勾选要导入的设备 → 完成

### 注意事项

- 米家集成需要网络访问小米云
- 设备数量多时，同步可能需要几分钟
- 如果部分设备不显示，检查是否在米家 APP 中已正常在线

## 二、海尔（Haier）⭐⭐⭐

海尔是所有品牌中**最难接入**的——它不用账号密码，而是用 Token 验证。需要用抓包工具获取。

### 安装步骤

#### A. 安装集成包

1. 下载海尔集成包（社区提供的 `haier` 文件夹）
2. 放入 `/docker/homeassistant/custom_components/haier/`
3. 重启 HA 容器
4. 设置 → 设备与服务 → 添加集成 → 搜索 `haier`

#### B. 抓取 Token（关键步骤）

海尔使用「令牌验证」而非密码，需要通过**小黄鸟抓包工具**来获取：

1. **安装小黄鸟抓包工具**（社区提供的网盘中可下载）
2. **电脑登录微信** → 搜索小程序「海尔智家」
3. **启动抓包**：打开小黄鸟，点击「调试」
4. **登录小程序**：用海尔智家 APP 的手机号登录
5. **停止抓包**：登录成功后点击停止
6. **找到 Token**：在小黄鸟右侧响应体中查找：
   - `id`：用户标识
   - `refreshToken`：刷新令牌
7. **回填**：将 `id` 和 `refreshToken` 粘贴到 HA 集成配置的对应框

### 小黄鸟抓包示意

```
响应体关键字段：
{
  "id": "xxxxxxxxxxxx",           ← 复制这个
  "refreshToken": "yyyyyyyyyyy",  ← 复制这个
  ...
}
```

> ⚠️ Token 有效期有限，过期后需要重新抓取。目前社区在推动海尔开放标准 OAuth 认证。

## 三、美的（Midea AC LAN）⭐

美的是最容易接入的品牌之一——HACS 里直接搜就能用。

### 安装步骤

1. **HACS 搜索**：HACS → 搜索 `Midea AC LAN` → 下载
2. **重启**：Container Manager 中重启 HA 容器
3. **添加集成**：设置 → 设备与服务 → 添加集成 → 搜索 `Midea AC LAN`
4. **登录**：输入「美的美居」APP 的账号 + 密码
5. **逐一添加**：设备逐个添加，完成一个再添加下一个

### 注意事项

- 如果装了 colmo（美的高端品牌）家电，先导入美的美居 APP，再通过 HA 集成接入
- 美的兼容 colmo，但 colmo 不兼容美的
- 多台空调需要逐一添加，不要跳过

---

## 四、海信（Hisense）⭐

海信是**最简单的**——只需要账号密码，且同时兼容海信和科龙品牌。

### 安装步骤

1. 下载海信集成包（社区提供 `hisense` 文件夹）
2. 放入 `/docker/homeassistant/custom_components/hisense/`
3. 重启 HA 容器
4. 设置 → 设备与服务 → 添加集成 → 搜索 `hisense`
5. 输入海信账号 + 密码
6. 选择家庭 → 选择设备 → 完成

### 兼容品牌

海信集成同时支持：
- 海信（Hisense）电视、空调
- 科龙（Kelon）空调

---

## 五、Apple HomeKit Bridge ⭐⭐

把 HA 中所有已接入的设备桥接到 Apple Home，实现 **Siri 语音控制**。

### 安装步骤

1. 设置 → 设备与服务 → 添加集成 → 搜索 `apple`
2. 选择 **HomeKit Bridge**
3. 默认配置 → 提交
4. 左下角通知栏 → 出现配对二维码
5. **iPhone 扫码**：相机扫码 → 或打开「家庭」App → 「+」→「添加或扫描配件」
6. 完成配对

### 效果

- 所有 HA 设备自动出现在 Apple Home 中
- 可以用 Siri 控制：「Hey Siri，关掉客厅灯」
- iPad / Mac / Apple Watch 也能控制
- 如果你有 HomePod，还能做家庭中枢，实现外网访问（无需花生壳）

## 集成后的效果

当所有品牌都成功接入后，你会看到类似这样的面板：

```
🏠 我的家
├── 💡 灯光
│   ├── 客厅主灯（米家）     💡 100%
│   ├── 餐厅吊灯（米家）     💡 80%
│   └── 卧室小夜灯（米家）   💡 10%
├── 🌡️ 空调
│   ├── 客厅空调（美的）     ❄️ 25°C
│   └── 主卧空调（colmo）    ❄️ 26°C
├── 📺 电视
│   ├── 客厅电视（海信）     ⏻ 关闭
│   └── 卧室电视（海信）     ⏻ 关闭
├── 🧊 冰箱
│   └── 厨房冰箱（海尔）     -18°C / 4°C
└── 🔒 安防
    └── 入户门锁（米家）     🔒 已锁
```

**一个界面，一个 APP，控制全家。**
